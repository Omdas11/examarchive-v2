# ExamArchive Logos

This directory contains logo assets used in the footer of ExamArchive.

## Platform Logos (64×64 PNG, Transparent)

Place the following logos in this directory:

- `github.png` - GitHub logo
- `google.png` - Google logo  
- `gemini.png` - Google Gemini logo
- `supabase.png` - Supabase logo
- `chatgpt.png` - ChatGPT/OpenAI logo

## University Logos (48×48 PNG, Transparent)

Place the following university logos in this directory:

- `assam-university.png` - Assam University logo
- `gauhati-university.png` - Gauhati University logo
- `tezpur-university.png` - Tezpur University logo
- `dibrugarh-university.png` - Dibrugarh University logo
- `iit-guwahati.png` - IIT Guwahati logo

## Requirements

- **Format:** PNG with transparent background
- **Platform Logos:** Recommended 64×64 pixels (displayed at ~56px)
- **University Logos:** Recommended 48×48 pixels (displayed at ~40px)
- **Optimization:** Use optimized PNGs to reduce file size

## Notes

The footer will gracefully handle missing logos using `onerror="this.style.display='none'"`, so the site will work even if logos are not yet added.
